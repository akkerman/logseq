#+title: Run slower to run faster

[[id:bac76b5d-85e1-49d2-9c25-295940a373b7][Je wordt een betere hardloper als je meer loopt.]] De enige manier om volume op te
bouwen zonder blessures is met lage intensiteit te rennen. Lage intensiteit
betekend daarom, zeker bij minder geoefende hardlopers, langzamer.

De manier om langzamer te lopen is letten op je hartslag. Het is namelijk
gebleken dat als je een  harloper een tempo doel geeft, men daar meestal
bovenuit wil gaan. Als je ze echter een hartslag grens geeft blijven ze daar
gemakkelijker onder.

Je traint een renner totdat ze minder snel moe worden, zodat ze genoeg
uithoudingsvermogen hebben om op hun natuurlijke snelheid te lopen gedurende
welke afstand ze ook lopen.
