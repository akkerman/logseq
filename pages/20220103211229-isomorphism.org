#+title: Isomorphism

In [[id:eb5dc1ed-3c0e-4b9d-aa23-00fa9639f583][Category Theory]] a function f: A→B is called an isomorphism if there exists a function g:B→A such that g.f = id_A and f.g = id_B



#+begin_src haskell :exports both
f :: Bool -> Int
f True  = 1
f False = 0

g :: Int -> Bool
g 1 = True
g 0 = False

(f . g) 0
(g . f) False
#+end_src
