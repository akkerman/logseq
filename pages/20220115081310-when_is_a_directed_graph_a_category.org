#+title: When is a directed graph a category

A [[roam:Graph]] is a [[id:65ef0e5c-8864-4f80-b92f-1c7ec0d560d2][Category]] when it has all identity and composition arrows.

- it is a directed graph
- the graph is [[id:b4330dd3-d7f2-4fe0-b262-e75e7ef3549c][reflexive]], each vertice has an edge to itself
- the graph is [[id:95e4a941-5ea0-4dcb-a18e-c133e60db46f][transitive]], if a → b and b → c then there is also a → c \\
