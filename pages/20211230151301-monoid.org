#+title: Monoid
#+options: toc:nil num:nil

A Monoid is a set with an [[id:615eca01-bd42-49fb-aa9a-0f981cea7674][associative binary operation]] with one special element which is neutral with respect to the operation.

This neutral element is also called a unit or identity element.

Examples:

|-----------------+-----------+-------|
| Set             | Operation | Unit  |
|-----------------+-----------+-------|
| Natural numbers | +         | 0     |
| Natural numbers | *         | 1     |
| Strings         | +         | ~""~    |
| Arrays          | concat    | ~[]~    |
| Sets            | union     | ~{}~    |
| Boolean         | and       | ~true~  |
| Boolean         | or        | ~false~ |
| Boolean         | xor       | ~false~ |
