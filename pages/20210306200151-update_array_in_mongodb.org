Voorbeelden hier beneden gaan over activiteten met een ~assignedTo~ array met 0 of meerdere organisatie namen.

** add to array

#+begin_src javascript
db.activiteit.updateMany({ /* some arbitrary query */ }, { $push: { "assignedTo": "GRONDG"  } })
#+end_src

** replace element in array

#+begin_src javascript
db.activiteit.updateMany({ "assignedTo": "NoNed" }, { $set: { "assignedTo.$": "GRONDG"  } })
#+end_src
