#+title: How do I create a note whose title already matches one of the candidates

In [[id:2103e6b5-7f31-4ee2-aafd-ab49f85fbc83][Org-roam]], when creating a note and a candidate is matched press [[id:a107ee75-8637-4d9a-86b7-a60f33b6ccbf][shortcut]] C-M-J to create a new one.

Zodra je de variable =ivy-use-selectable-prompt= zet, kun je de regel (met je invoer) selecteren met pijltje omhoog.
