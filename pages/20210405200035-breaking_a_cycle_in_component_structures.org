#+title: Breaking a cycle in component structures
#+options: toc:nil

Volgens [[id:a2a69177-50c1-4dee-ad4d-c14d61006db5][ADP]] moet je cycles in je component structuur voorkomen, als je er toch één hebt dan heb je de volgende twee methodes om de cycle weg te halen.

* Invert the dependency
:PROPERTIES:
:ID:       ca49c8ac-c555-4f1d-a3f7-282601f40710
:END:

In het component dat de afhankelijkheid heeft, maak een interface die geimplementeerd wordt door de component waarvan je afhankelijk bent. Nu is de afhankelijkheid omgedraait en de cycle weg.
